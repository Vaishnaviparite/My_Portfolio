Some of My Recent Works

IoT-Based Vehicle Accident Detection

This project helps detect road accidents automatically and sends alerts to emergency contacts. It uses sensors to detect crashes, a GPS module to find the accident location, and a GSM module to send messages. This system ensures quick help and can save lives by reducing emergency response time.

Hospital Management System

This project is a web-based system designed to manage hospital operations efficiently. It uses React for the frontend, Node.js for the backend, and MongoDB as the database. The system allows doctors, patients, and staff to manage appointments, medical records, and billing in one place. With a user-friendly interface, it helps hospitals streamline workflows and improve patient care by making data accessible and well-organized.

Fingerprint-Based Door Lock System Using IoT

This project improves security by using fingerprints instead of keys or passwords. The system scans a fingerprint, checks if it's authorized, and unlocks the door. It also keeps a record of who enters. This makes access safer and more convenient, perfect for homes and offices.
